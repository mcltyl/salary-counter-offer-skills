# Salary Counter-Offer 💰

**You got the job offer. Now what? A step-by-step framework for negotiating salary without burning bridges.**

Most people leave money on the table because they don't know what to say. This skill gives you the exact words.

---

## The Problem

You got the offer. Now you're thinking:
- "Should I even negotiate?"
- "What if they rescind the offer?"
- "What number do I ask for?"
- "What do I actually say?"

**The truth:** Negotiation is expected. Not negotiating is the mistake.

## The Core Insight

| ❌ What You Fear | ✅ Reality |
|------------------|-----------|
| They'll rescind the offer | Almost never happens |
| They'll think I'm greedy | They'll think you know your worth |
| First offer is fair | First offer is their starting point |
| I should be grateful | Gratitude and negotiation coexist |
